# Changelog

All notable changes to **IGLookup** will be documented in this file.

## [1.0.0] - 2026-05-27

### Added
- Initial release
- Fetch Instagram account info by username
- Display last 5 posts (caption, likes, comments, URL)
- Colorful terminal UI with ANSI colors
- Loading animation
- Auto-save results to `output/` as timestamped JSON
- Cross-platform support (Linux, Windows, Mac)
